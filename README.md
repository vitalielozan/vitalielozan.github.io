# Vitalie Lozan Portfolio

## English

Personal portfolio built with React, Vite, Tailwind CSS, and Framer Motion.
It presents selected projects, technical skills, and professional background in a clean, responsive interface.

Live site: <https://vitalielozan.github.io>

### Stack

- React
- Vite
- Tailwind CSS
- Framer Motion

### Scripts

- `npm install` - install dependencies
- `npm run dev` - start local development
- `npm run build` - create production build
- `npm run deploy` - deploy to GitHub Pages
